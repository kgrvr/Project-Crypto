About Project

The “Crypto” is a Java based Web Application that can generate cipher codes to secure data in a coded form. Developing this application aims at providing a user-friendly environment for encoding and decoding of the data.

Crypto (abbreviated as Cryptography) is used to secure the information. In the simple words it is the art of writing and solving codes. More generally, cryptography is about constructing and analyzing protocols that prevent third parties or the public from reading private messages; various aspects in information security such as data confidentiality, data integrity, authentication, and non-repudiation are central to modern cryptography. Modern cryptography exists at the intersection of the disciplines of mathematics, computer science, and electrical engineering. Applications of cryptography include ATM cards, computer passwords, and electronic commerce.

We have used Java Server Pages (JSP), JavaScript, Cascading Style Sheets (CSS), and HyperText Markup Language (HTML) as frontend and Java as backend to build our project. Our project contains different modules working together to form a system that work efficiently and effectively.

It currently supports following Encryption Techniques:

  • Substitution Scheme
    1. Caeser Cipher Encryption Technique
    2. Vernam Cipher Encryption Technique

  • Transposition Scheme
    1. Hill Cipher Encryption Technique
    2. PlayFair Cipher Encryption Technique
